# Python Log Monitoring Project


## Objective
A Python-based log monitoring and analytics tool.

## Requirements
1. Read logs from the file
2. Identify log levels: INFO, WARNING, ERROR
3. Provide analytics:
   - Total log count
   - Count per log level
   - Most frequent ERROR message
   - Filter logs by time range
4. Provide a CLI-based menu
5. Export analytics to a JSON report

## Clone && Running 

1) Clone the repository:

    - git clone https://github.com/Sandeepreyyi/log_monitor.git
    - cd log_monitor

2) Log file Set up:

    - Add log file in log_monitor folder
    - change the log file name in main.py file 


3) Run the application:

    python main.py
